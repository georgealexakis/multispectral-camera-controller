Multispectral camera controller

Project github page: github.com/georgealexakis/multispectral-camera-controller

Experimental application for multispectral cameras and ROS. It connects with ROS client API for camera reconfiguration. You need to install rosbridge suite to enable the connection between ROS and the controller. This controller only has been tested with ROS melodic.

The project structure is good for using this with Apache cordova and build a hybrid application. This project enables the reconfiguration of camera parameters such as exposure, frame rate etc. Especially this project is used for multispectral camera with 9 bands.

This is the command to install rosbridge_suite:
sudo apt-get install ros-rosdistro-rosbridge-server

This is the github page of rosbridge: github.com/RobotWebTools/rosbridge_suite
